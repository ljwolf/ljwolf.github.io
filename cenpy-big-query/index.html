Cenpy Big Query

If you’ve been using my tools for census data exploration, as intensely as I’d hope you have, you might’ve noticed that the Census limits queries to 50 columns per query.

I’ve got your back, though. The newest release, cenpy 0.7, now supports n-column queries by chunking through 50-coulumns at a time. Be careful, though, as I’m sure if you pull down the entire Census 2010 short form in one go, you will probably get rate limited.
